# Página de Restaurantes en Tepic con API de Google Maps

# Descripción: Proyecto personal es una página web donde se pueden buscar restaurantes de mi ciudad, los datos se obtienen mediante una API y se realizan filtraciones dinámicas usando botones o el buscador 
Aún se encuentra en desarrollo pues faltan áreas como el login y las páginas de recomendaciones, pero el proyecto ya muestra manejo de datos.

# Tecnologías:
- HTML
- SASS
- JavaScript
- API externa para los restaurantes

# Funcionalidades implementadas
- Búsqueda de restaurantes con filtrado dinámico.
- Visualización de resultados de manera interactiva.
- Diseño responsivo parcial usando Flexbox y Grid.
- Modularización de CSS con Sass.

# Notas
Este proyecto refleja mi capacidad para desarrollar **aplicaciones web interactivas y dinámicas**, y mi disposición a aprender y completar funcionalidades faltantes para lograr un producto final sólido.
